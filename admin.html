Setup Required: GitHub Token

To save content directly from this panel, you need a GitHub Personal Access Token:

  1. Go to github.com/settings/tokens/new
  2. Name it HarpShed Admin
  3. Set expiration to 90 days (or no expiration)
  4. Under "Repository access" select "Only select repositories" and choose harpshed-site
  5. Under Permissions > Repository permissions, set Contents to "Read and write"
  6. Click "Generate token" and paste it in Settings below

Until then, you can still edit content and download the file manually.
